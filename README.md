# Rock Paper Scissors Game 

A fun and interactive **Rock Paper Scissors** game built with **HTML, CSS, and JavaScript**.  
This project helped me practice **DOM manipulation**, **conditional logic**, and **localStorage** for persisting scores.

## Features 
- Play Rock, Paper, or Scissors against the computer  
- Track your **Wins, Losses, and Draws** across sessions  
- **Reset scores** anytime  
- Responsive and styled user interface  

## How to Play 
1. Clone or download this repository  
2. Open `RockPaperScissors.html` in your browser  
3. Click on **Rock**, **Paper**, or **Scissors** buttons to play  
4. See the results and scores live on the page  

## Tech Stack 
- HTML5  
- CSS3  
- JavaScript
